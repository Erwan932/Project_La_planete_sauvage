# Project_La_planete_sauvage

La Fugue d'Agile
Plateformer - Infiltration 2D
Niveau tutoriel
Manette recommandée
Moteur de Jeu : Unity

Contrôles :
(Modèle Manette Xbox)

Joystick gauche : Déplacement
Start : Menu Pause
A : Sauter
B : Siffler pour briser les cristaux
Y : Passer les dialogues
X : Libérer les prisonniers


Synopsis : Ygam est une planète hostile pour les Hommes. 
Ils ne sont que l’équivalent d’insectes intelligents pouvant vaguement s'organiser.
L'espèce maître de la planète, les Draags, sont des humanoïdes privilégiant le savoir avant tout.
Mais ils n'ont que très peu d'estime pour la faune de leur planète.
En effet, les Hommes sont souvent réduits à être des animaux de compagnie pour les grands de ce monde.​
C'est là qu'interviennent les Hommes sauvages, souhaitant libérer leurs congénères, les Hommes de luxe. 
Ils vont s'infiltrer dans des lieux de vie des Draags, libérer et s'enfuir en groupe. 
Mais la tâche n'est pas aisée. Les Draags sont immenses et de nombreux obstacles se dressent sur le trajet.​
Tentez d'emmener vos semblables vers la liberté, mais attention à ne pas vous faire prendre.
